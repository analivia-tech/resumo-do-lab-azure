# resumo-do-lab-azure
Este repositório contém o resumo das lições aprendidas durante o desenvolvimento do lab na DIO (A Z Azure)
Não há diferenciação entre a interface gratuita e a paga, a diferença pode ser em relação aos recursos criados.
A plataforma pode ser costumizada conforme o gosto do usuário (cores, idiomas, formatação dos menus etc.)
Barra lateral à esquerda informa as [principais] funcionalidades.
Bastions: Rede segura para fazer o acesso às máquinas.
Armazenamento: Possui setor de migração de armazenamento.
Prestar atenção quanto à versão prévia, que é quando o serviço está começando a ser disponibilizado, pois existe a possibilidade de o produto ser retirado de uso. Versão prévia não é garantida a permanência.
